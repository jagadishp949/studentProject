

install Python 3.5.3  and all others required packages mentioned in requirements.txt file for studentproject, please check word file(compilations) for i/p and o/p.

Steps for creating a project in visualStudio for promblem 5:

1)under file->new->Project
2) select python->Django Web Project
3) in location give the project folder directory,  in Name and SolutionName give the name studentProject
4)click on ok ->project will be created. 


 or you can directly run the command "py manage.py runserver" in studentProject to start the app of student project.
